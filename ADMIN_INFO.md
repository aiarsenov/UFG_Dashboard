# Информация об админской учетной записи

## Админский email

**Админская учетная запись:** `vasiliy_arsenov@bizan.pro`

Этот email настроен в переменной окружения `ADMIN_EMAIL` и используется для доступа к админ-панели (`/admin`).

## Как получить доступ к админке

1. Зарегистрируйтесь или войдите с email: `vasiliy_arsenov@bizan.pro`
2. Перейдите на `/admin` - должна открыться админ-панель
3. В админке вы можете:
   - Создавать новых пользователей
   - Просматривать список всех пользователей

## Изменение админского email

Если нужно изменить админский email:

1. **Локально:** Обновите `.env.local`:
   ```
   ADMIN_EMAIL=новый@email.com
   ```

2. **В Vercel:** Обновите переменную окружения `ADMIN_EMAIL`:
   ```bash
   echo "новый@email.com" | vercel env add ADMIN_EMAIL production
   ```

3. **Или измените в коде:** Отредактируйте `src/lib/config.ts` и замените дефолтное значение.

## Несколько админов

Можно указать несколько email через запятую:
```
ADMIN_EMAIL=admin1@example.com,admin2@example.com
```
